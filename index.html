about me

Halo, saya Sindu. Saya seorang pemula dalam dunia pengembangan web dan sedang belajar untuk menjadi seorang web developer. Saya memiliki semangat dan ketertarikan yang besar dalam dunia coding dan desain web. Saat ini, saya sedang memperdalam pengetahuan saya dalam HTML, CSS, dan JavaScript. Saya sangat antusias dalam menghadapi tantangan baru dan senang terlibat dalam proyek-proyek web yang kreatif. Saya siap untuk terus belajar dan meningkatkan kemampuan saya sebagai seorang web developer.

Saya mengadopsi pendekatan belajar yang proaktif dan mandiri. Saya aktif dalam mencari sumber belajar online, seperti tutorial, dokumentasi, dan kursus interaktif. Saya juga bergabung dalam komunitas pengembang web untuk berbagi pengetahuan dan belajar dari sesama
